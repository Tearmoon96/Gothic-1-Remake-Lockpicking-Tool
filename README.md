# Gothic-1-Remake-Lockpicking-Tool
Since lockpicking is stupidly hard i made this tool. Pistons are counted from left to right and plates from top to bottom. By inputting the starting positions of the lock plates and configuring their behaviors (how moving one plate affects the others: "Same", "Opposite", or "None"), the tool calculates the optimal sequence of moves

Each folder contains the respective `.py` source code and a `requirements.txt` file to install the required dependencies.

Neither of the two versions ( windows or linux ) inject anything into the game or the system. If you don't trust the .exe review the python code and build your own
